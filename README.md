# Supplier Performance Dashboard (Power BI)

## This project analyzes supplier performance using key KPIs:
- Average Lead Time
- Late Deliveries
- Total Orders and Quantities
- Supplier Comparison by Performance

## Tools Used
- Microsoft Power BI Desktop
- DAX for KPI calculations
- Dataset: Procurement KPI Analysis Dataset

## Description
The dashboard provides insights into supplier reliability and delivery performance, identifying top-performing and underperforming suppliers based on lead times and delivery consistency.
